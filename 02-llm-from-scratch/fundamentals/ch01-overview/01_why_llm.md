# ch01-01: 为什么学习大模型

## 大模型时代

2022年ChatGPT发布后，大语言模型(LLM)成为AI领域的核心技术。从GPT到Claude，从Gemini到Llama，大模型正在改变人机交互的方式。

## 为什么要从零理解

| 学习方式 | 优点 | 缺点 |
|----------|------|------|
| 调用API | 简单快捷 | 不知道内部原理 |
| **从零学习** | 理解本质 | 需要较多时间 |

**从零学习的价值：**
- 理解Transformer为什么有效
- 掌握训练优化的关键技术
- 为深入研究打下基础

## 大模型训练的核心流程

```
1. 预训练 (Pretraining)
   海量文本 → 语言模型 → 通用能力
   
2. 微调 (Fine-tuning)
   特定数据 → 任务能力
   
3. 对齐 (Alignment)
   人类反馈 → 符合人类价值观
```

## 本教程的目标

| 进阶 | 掌握预训练、微调、RLHF |
