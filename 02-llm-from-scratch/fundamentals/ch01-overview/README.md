# 从零理解大模型 - 学习教程

## 教程简介

本教程从零开始，系统讲解大语言模型（LLM）的核心原理与训练方法。通过理论学习+代码实践的方式，深入理解Transformer架构、预训练、微调等关键技术。

## 如何使用本教程

### 推荐学习顺序

1. **快速浏览**：通读各章节README，了解整体框架
2. **理论学习**：阅读Markdown文档，理解核心概念
3. **代码实践**：运行示例代码，加深理解
4. **项目实战**：完成课程项目，巩固知识

### 每章的结构

```
chXX/
├── README.md          # 章节介绍和内容概览
```
