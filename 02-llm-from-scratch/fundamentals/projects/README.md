## 项目列表

### 1. 垃圾短信分类器

**目录**: `spam-classifier/`

**技术**: GPT-2 微调、类别权重、冻结参数

**成果**: 
- 准确率: 96.7%
- F1 分数: 87.6%

**学习要点**:
- 如何微调预训练模型
- 解决数据不平衡问题
- 冻结策略的应用

**运行**:
```bash
cd spam-classifier
python train.py    # 训练
python predict.py  # 预测
```
