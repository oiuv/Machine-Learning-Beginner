# 诡秘之主 GPT 项目

基于《诡秘之主》小说训练的中文 GPT 模型。

---

## 项目概述

使用完整的《诡秘之主》小说（约1440万字符）训练一个能生成小说风格中文文本的 GPT 模型。

---

## 硬件配置

| 配置项 | 值 |
|-------|-----|
| GPU | NVIDIA RTX 4090 Laptop |
| 显存 | 16 GB |
| 模型 | GPT-2 Small (124M) |
| Batch Size | 16 |
| 预计训练时间 | 15-25 分钟 |

---

## 模型配置

```python
{
    "vocab_size": 50000,      # 中文 BPE 词表
    "context_length": 512,    # 上下文长度
    "emb_dim": 768,          # 嵌入维度
    "n_heads": 12,           # 注意力头数
    "n_layers": 12,          # Transformer 层数
    "batch_size": 16,
    "learning_rate": 5e-4,
    "num_epochs": 3
}
```

---

## 训练流程

### 阶段1: 数据预处理
- 加载《诡秘之主》txt 文件
- 清洗和分割文本
- 统计：约1440万字符，12万行

### 阶段2: 训练 BPE 分词器
- 训练中文 BPE 分词器
- 词表大小：50,000 tokens
- 保存分词器配置

### 阶段3: 创建数据集
- 滑动窗口生成训练样本
- 划分训练/验证集 (95%/5%)

### 阶段4: 创建模型
- 初始化 GPT-2 模型
- 适配中文词表
- 参数量：约124M

### 阶段5: 训练
- 3 epochs 训练
- 监控训练/验证损失
- 保存最佳模型

## 使用方法

### 1. 训练模型

```bash
cd learning/projects/guimi-gpt
python train.py
```

### 2. 生成文本

```bash
# 演示模式
python generate.py

# 交互模式
python generate.py -i

# 单次生成
python generate.py "克莱恩看着面前的笔记本" 200
```

## 预期效果

训练完成后，模型能够：

1. **续写小说**
   ```
   输入: 克莱恩看着面前的笔记本
   输出: 克莱恩看着面前的笔记本，那上面记载着...
   ```

2. **生成小说风格文本**
   ```
   输入: 绯红的月光照耀着廷根市
   输出: 绯红的月光照耀着廷根市，街道上弥漫着...
   ```

## 技术要点

### 1. 中文 BPE 分词
- 使用 tokenizers 库训练 BPE
- 针对中文优化
- 词表大小 50,000

### 2. GPT-2 架构
- 12 层 Transformer
- 12 个注意力头
- 768 维嵌入

### 3. 训练优化
- AdamW 优化器
- 学习率 5e-4
- 梯度累积（如需要）

---

## 注意事项

1. **显存管理**
   - batch_size=16 约占用 4-6GB
   - 可根据显存情况调整

2. **训练时间**
   - 首次训练 BPE 分词器需要 2-5 分钟
   - 模型训练约 15-25 分钟

3. **效果**
   - 3 epochs 后模型能生成基本流畅的中文
   - 更多 epochs 可能提升效果
