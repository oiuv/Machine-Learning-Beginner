# 指令微调项目

基于 GPT-2 的中文指令微调实践项目。

---

## 项目目标

训练一个能理解和执行指令的模型，例如：
- 翻译： "把 Hello 翻译成中文" → "你好"
- 摘要： "总结这段文字" → "简短摘要"
- 问答： "回答问题" → "答案"

---

## 数据集

**50条中文指令**，包含9种任务类型：

| 任务类型 | 数量 | 示例 |
|---------|------|------|
| 翻译（中→英） | 5 | "今天天气很好" → "The weather is nice today" |
| 翻译（英→中） | 5 | "Hello world" → "你好世界" |
| 摘要 | 5 | 长文本 → 简短总结 |
| 问答 | 5 | 问题 → 答案 |
| 改写 | 5 | 粗鲁 → 礼貌 |
| 情感分类 | 5 | 评论文本 → 正面/负面/中性 |
| 生成 | 5 | 主题 → 句子 |
| 概念解释 | 5 | 术语 → 定义 |
| 句子改写 | 5 | 陈述句 → 疑问句 |
| 信息提取 | 5 | 文本 → 关键信息 |

---

## 技术要点

### 1. Prompt Template

```
### 指令:
{instruction}

### 输入:
{input}

### 回答:
{output}
```

### 2. Loss Masking

- **只计算 Response 部分的损失**
- Instruction 和 Input 部分设为 `-100`（忽略）

### 3. 训练设置

```
模型: GPT-2 (124M)
批次大小: 4
学习率: 5e-5
训练轮数: 3
优化器: AdamW
```

## 使用方法

### 1. 训练模型

```bash
cd learning/projects/instruction-finetuning
python train.py
```

**预计时间**: 10-20 分钟（取决于 GPU）

### 2. 测试模型

```bash
# 演示模式
python predict.py

# 交互模式
python predict.py -i
```

## 预期效果

### 训练前（预训练 GPT-2）

```
输入: ### 指令: 把Hello翻译成中文 ### 输入: Hello ### 回答:
输出: ### 指令: 把Hello翻译成中文 ### 输入: Hello ### 回答: 请问您需要...
                                                    ↑ 只是续写
```

### 训练后

```
输入: ### 指令: 把Hello翻译成中文 ### 输入: Hello ### 回答:
输出: 你好
          ↑ 正确执行了翻译任务
```

---

## 学习要点

1. **指令数据格式化**：instruction + input → output
2. **Prompt Template**：统一格式，让模型识别任务
3. **Loss Masking**：只训练 Response 生成
4. **多任务学习**：同时训练多种任务，提升泛化能力

---

## 注意事项

1. **数据量少**：只有50条样本，效果可能不完美
2. **英文模型**：GPT-2 是英文模型，中文效果可能有限
3. **改进方向**：
   - 增加数据量（1000+条）
   - 使用中文预训练模型（如 Chinese-GPT-2）
   - 延长训练时间（10+ epochs）
