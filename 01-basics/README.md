# 🌱 基础篇：机器学习入门

本章节完全从零手写代码，不使用任何机器学习框架，帮助你直观理解机器学习的核心概念。

## 📚 章节导航

### [01-linear-model](01-linear-model/) - 从 `y=wx` 开始
**核心概念：**
- 机器学习与传统编程的区别
- 模型、参数、特征、标签
- 学习率的作用
- 欠拟合、过拟合、发散

**你将实现：** 最简单的线性模型 `y = wx`，观察机器如何学习参数 `w`

### [02-gradient-descent](02-gradient-descent/) - 梯度下降
**核心概念：**
- 均方误差损失函数
- 梯度下降算法
- 前向传播和反向传播
- 导数和偏导数

**你将实现：** 使用梯度下降训练模型，理解误差如何反向传播

---

### [03-activation](03-activation/) - 激活函数
**核心概念：**
- 为什么需要非线性变换
- Sigmoid、ReLU、Tanh 等激活函数
- 激活函数对模型表达能力的影响

**你将实现：** 对比线性和非线性模型，感受激活函数的作用

---

### [04-neural-network](04-neural-network/) - 神经网络
**核心概念：**
- 神经元和隐藏层
- 多层前馈神经网络
- 反向传播在深层网络中的传播
- 网络深度和宽度的权衡

**你将实现：** 一个完整的多层神经网络，理解深度学习的本质
