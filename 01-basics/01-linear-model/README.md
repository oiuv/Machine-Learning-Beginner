# 01-linear-model: 从 `y=wx` 开始

> 本节内容你可以了解机器学习和传统编程的差别，了解机器学习中参数、特征、标签等概念，并通过最简单的模型 `y=wx` 入门机器学习。

## 📖 本节内容

### 1. 传统编程 vs 机器学习
- 传统编程：明确规则 → 输出
- 机器学习：数据 + 输出 → 学习规则

### 2. 核心概念
- **模型**：`y = wx`（最简单的线性模型）
- **参数**：`w`（机器需要学习的值）
- **特征**：`x`（输入数据）
- **标签**：`y`（期望输出）
- **学习率**：控制参数更新的步长

### 3. 学习过程可视化
- 观察 `w` 如何逐步逼近真实值
- 理解学习率过大/过小的问题
- 认识欠拟合、过拟合、发散

## 🚀 运行代码

```bash
jupyter notebook tutorial.ipynb
```

## 📝 代码结构

```python
# 伪代码展示学习过程
w = 随机初始化
for 迭代次数:
    预测 = w * x
    误差 = 真实值 - 预测
    w = w + 学习率 * 误差 * x  # 参数更新
```

## 🎯 课后练习

1. 修改学习率，观察收敛速度的变化
2. 尝试不同的初始 `w` 值
3. 添加噪声数据，观察模型的鲁棒性
4. 尝试 `y = wx + b`（带偏置项）

## 💡 关键理解

> 机器学习就是让机器通过数据自动找到最优的参数，使得模型的预测尽可能接近真实值。
