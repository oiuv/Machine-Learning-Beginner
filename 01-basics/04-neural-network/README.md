# 04-neural-network: 手撸神经网络

> 手撸神经网络感受深度学习，理解神经元、隐藏层、前向传播和反向传播的完整过程。

## 📖 本节内容

### 1. 神经网络架构
- 输入层、隐藏层、输出层
- 神经元和连接权重
- 偏置项的作用

### 2. 前向传播
```
h = activation(W₁x + b₁)  # 隐藏层
y = W₂h + b₂              # 输出层
```

### 3. 反向传播
- 计算输出层梯度
- 梯度反向传播到隐藏层
- 链式法则的应用
- 参数更新

### 4. 网络设计
- 隐藏层数量 vs 每层神经元数量
- 宽度和深度的权衡
- 过拟合和正则化

## 💡 关键理解

> 神经网络的本质是：通过多层非线性变换，将输入数据映射到输出空间。每一层提取不同层次的特征，深层网络能够学习更抽象、更复杂的模式。
